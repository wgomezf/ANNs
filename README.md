# `An introduction to artificial neural networks for biomedical data`

This workshop explores the basics of Artificial Neural Networks (ANNs) by introducing essential concepts, such as the perceptron model, and traces the evolution to more complex architectures, including multilayer perceptrons and Convolutional Neural Networks (CNNs). Participants will also gain insights into the mathematical principles underlying neural network training, including backpropagation, optimization algorithms, regularization techniques, and GRAD-CAM methods for improving model explainability. A significant focus of the workshop is the application of ANNs and CNNs to classify breast lesions in ultrasound scans through practical exercises in Python on Google Colab, featuring "from scratch" implementations that bridge mathematical theory with programming practice, along with the use of key libraries for developing artificial neural networks.
